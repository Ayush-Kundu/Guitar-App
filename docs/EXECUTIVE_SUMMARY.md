# Guitar App - Executive Summary for Publishers

**One-Page Overview | January 2026**

---

## What We Are

An **interactive guitar learning app** that teaches users to play songs through visual guidance and real-time synthesized audio.

## What We Are NOT

- ❌ A music streaming service
- ❌ A sheet music publisher
- ❌ A MIDI/audio download platform

---

## The Core Difference

| Traditional Music Apps | Guitar App |
|------------------------|------------|
| Store and play audio files | Synthesize audio in real-time |
| Users can extract/share songs | No extractable content exists |
| Passive listening possible | Active practice required |
| Sheet music/tabs provided | Visual fretboard guidance only |

---

## Technical Safeguards

### ✅ What's Enabled
- Real-time synthesized audio (no files)
- Tempo adjustment (50%-150%)
- Section looping for practice
- Visual fretboard guidance
- Progress tracking

### ❌ What's Permanently Blocked
- Audio export (WAV/MP3)
- MIDI export
- Sheet music generation
- Tablature text export
- Full-song autoplay
- Clipboard copy of note data

**"Can users get the song out?" → PROVABLY NO**

---

## Content Policy

| Content Type | Status |
|--------------|--------|
| Public Domain | ✅ Included (Greensleeves, Bach, etc.) |
| Original | ✅ Created by us |
| Licensed | ✅ Only with explicit written permission |
| Unlicensed | ❌ Technically blocked - cannot load |

Every song file requires valid license metadata. Our validator **rejects** songs without proper licensing.
